## 🎨 Features

- Fully responsive layout (mobile + desktop)
- Hero section with café intro
- About section with poetic description
- Interactive Menu with categories
- Image gallery
- Contact information
- Soft colors and custom fonts
- Hosted using GitHub Pages

---

## 📦 Tech Stack

- HTML5
- CSS3 (no frameworks)
- Google Fonts
- Git & GitHub Pages
